## Ключевые моменты
### <a name="Глава_1._Первые_шаги._Не_теряя_времени"></a> Глава 1. Первые шаги. Не теряя времени

- Ключевое слово var определяет переменную, значение которой может **изменяться**;
- Ключевое слово val определяет переменную, значение которой остается **неизменным**;
- println вставляет новую строку после вывода, а print продолжает вывод в той же строке. 

#### <a name="Базовые_типы_и_переменные._Из_жизни_переменных"></a> Базовые типы и переменные. Из жизни переменных

Тип | Наименование | Биты | Диапазон значений |
----|------|-------------------|-------------------|
Целочисленный | Byte | 8 битов | –128 до 127 |
Целочисленный | Short | 16 битов | –32768 до 32767 |
Целочисленный | Int | 32 бита | –2147483648 до 2147483647 |
Целочисленный | Long | 64 бита | –huge до (huge - 1) |
С плавающей точкой |  Float | 32 бита | -3.4*1038 до 3.4*1038 |
С плавающей точкой |  Double | 64 бита | ±5.0*10-324 до ±1.7*10308 |
Логический | Boolean | - | true или  false |
Символы и строки | Char | 16 битов | 1 символ |
Символы и строки | String | - | наборов символов, объединенных в цепочку |

- Объявление переменной с явным указанием типа, выглядит так: var smallNum: Short;
- В переменной хранится ссылка на объект;
- Преобразование переменной из одного типа в другой, происходит следующим образом: var z: Long = **x.toLong()**;
- Массивы создаются функцией arrayOf(). Например var myArray = arrayOf(1, 2, 3). Для получения размера массива используется функция: myArray.size;
- Math.random() возвращает случайное число в диапазоне от 0 до (почти) 1;
- Явное определение типа массива var myArray: Array<Byte> = arrayOf(1, 2, 3);
- При указании val можно обновлять переменные в массиве.

### <a name="Глава_2._Функции._За_пределами_main"></a> Глава 2. Функции. За пределами main

- Создание функции выглядит следующим образом:
```java  
fun foo(int1: Int, int2: Int) {
 println("Parameter is $param")
}
``` 
- Вызов функции выглядит следующим образом:
```java  
fun main(args: Array<String>) {
 foo()
}
``` 
- Функции из единственного аргумента выглядят следующим образом:
```java  
fun max(a: Int, b: Int): Int = if (a > b) a else b
``` 
 - Объявление возвращаемого типа **Unit** означает, что функция не возвращает значения, в ином случае возвращает  значение объявленного типа;

- Цикл выглядит следующим образом:
```java  
for (x in 1..10) {
 //Здесь размещается код
}
``` 
- Оператор **until** исключает коечное число из цикла, так, представленный ниже цикл выводит числа от 1 до 99. Оператор .. выведет числа от 1 до 100
```java  
for (x in 1 until 100) println(x)
``` 
- Оператор **downTo** выведет числа в обратном порядке:
```java  
for (x in 15 downTo 1) println(x)
``` 
- Оператор **step** задает размер шага:
 ```java  
for (x in 1..100 step 2) println(x)
``` 
- Перебор элементов массива по индексам:
```java  
for (index in optionsParam.indices) {
    println("Index $index has item ${optionsParam[index]}")
}
``` 
- Цикл перебирающий все элементы массива. Индекс элемента присваивается переменной index, а сам элемент — переменной item;
```java 
for ((index, item) in optionsParam.withIndex()) {
    println("Index $index has item $item")
}
```
- Операторы ‘И’ и ‘Или’ (&& и ||)
-  Kotlin существуют функции toLowerCase , 
toUpperCase и capitalize для создания строки, преобразованной к нижнему регистру, верхнему регистру или к написанию 
с начальной прописной буквы. Например, следующая команда 
использует функцию capitalize для преобразования первой 
буквы строки с именем userInput:
```java 
 userInput = userInput.capitalize()
```
#### <a name="Классы_и_объекты._Высокий_класс"></a> Классы и объекты. Высокий класс
- Определение класса выглядит следующим образом:
```java
сlass Dog (val name: String, var weight: Int, val breed: String) {
    fun bark() {
               println(if (weight < 20) "Yip!" else "Woof!")
               }
}
```
- Функция, определенная 
внутри класса, называется функцией класса. 
Также иногда используется термин «метод».
- Массивы объектов задаются следующим образом:
```java
var dogs = arrayOf(Dog("Fido", 70, "Mixed"), Dog("Ripper", 10, "Poodle"))
```
- Объявление объекта с присваиванием состоит из трех 
основных этапов:
1. Объявление переменной
```java
var myDog = Dog("Fido", 70, "Mixed")
```
2. Создание объекта
3. Связывание объекта с переменной посредством присваивания 
ссылки.
- Конструктор выполняется при создании 
экземпляра объекта. 
Он используется для 
определения свойств 
и их инициализации.
- Представление конструктора
При создании класса Dog мы включаем конструктор — круглые скобки 
и заключенный в них код — в заголовок класса:
```java
class Dog(val name: String, var weight: Int, val breed: String) {
 ...
}
```
- Объект иногда называется
экземпляром класса, а его 
свойства — переменными
экземпляров
- Гибкая инициализация свойств
Допустим, вы хотите добавить в класс Dog свойство activities и 
инициализировать его массивом, который по умолчанию содержит значение «Walks». Код, который это делает, выглядит так:
```java
class Dog(val name: String, var weight: Int, val breed: String) {
    var activities = arrayOf("Walks")
 ...
}
```
- А может оказаться, что вы хотите изменить значение параметра 
конструктора перед тем, как присваивать его свойству. Допустим, 
свойству breed вместо значения, переданного конструктору, 
должна присваиваться версия строки, преобразованная к верхнему регистру. Для этого функция toUpperCase создает версию 
строки, преобразованную к верхнему регистру, которая затем 
присваивается свойству breed:
```java
class Dog(val name: String, var weight: Int, breed_param: String) {
    var activities = arrayOf("Walks")
    val breed = breed_param.toUpperCase()
 ...
}
```
- *Блоки инициализации*   
Если свойство должно инициализироваться чем-то более сложным, чем простое выражение, или если при создании каждого объекта должен выполняться дополнительный 
код, можно использовать один или несколько блоков инициализации. Блоки инициализации выполняются при инициализации объекта сразу же после вызова конструктора 
и снабжаются префиксом init. Следующий блок инициализации выводит сообщение 
каждый раз, когда инициализируется объект Dog:
```java
class Dog(val name: String, var weight: Int, breed_param: String) {
    var activities = arrayOf("Walks")
    val breed = breed_param.toUpperCase()
    init {
       println("Dog $name has been created.")
 }
 ...
}
```

- Наследование предотвращает дублирование кода;
- Класс, содержащий общий код, называется суперклассом, а классы, наследующие от него, называются подклассами.
