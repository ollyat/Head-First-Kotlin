# Phrase-O-Matic

Мы создадим новое приложение, генерирующее эффектные 
рекламные слоганы.
Начните с создания нового проекта в IntelliJ IDEA:

1. Откройте IntelliJ IDEA и выберите команду «Create New Project» на заставке. 
При этом запускается программа-мастер, описанная в главе 1.

2. Когда мастер выдаст соответствующий запрос, выберите настройки создания 
проекта Kotlin, предназначенного для JVM.

3. Присвойте проекту имя «PhraseOMatic», подтвердите остальные значения 
по умолчанию и щелкните на кнопке Finish.

4. Когда проект появится в IDE, создайте новый файл Kotlin с именем PhraseOMatic.
kt — выделите папку src, откройте меню File и выберите команду New > Kotlin 
File/Class. Когда будет предложено, введите имя файла «PhraseOMatic» и выберите вариант File в группе Kind.
